Directive definition object properties

Notice: All example directives are located in /directives/directive_examples/ folder and are put in directives.html file.

  1. template: 'html string' - template used as a HTML string [01template-string.js]
  2. template: function(tElement, tAttrs) { ... } - template used as a function which returns HTML string [01template-func.js]
  3. templateUrl: 'mytemplate.html' - template URL is using HTML file in /dist/ folder [02templateurl.js]
  4. templateUrl: function(tElement, tAttrs) { ... } - template used as a function which returns template URL [02templateurl-func.js]
    NOTICE: In this example template is defined inside JS file (main.js) using $templateCache. 
    //define default templates
clientApp.run(function ($templateCache) {
    'use strict';
    $templateCache.put('green.html', '<p style="color:green">green template</p>');
    $templateCache.put('red.html', '<p style="color:red">red template</p>');
});
  5. controller: 'someCtrl' - external controller is used [03controller-external.js, directivesCtrl.js]
  6. controller: function($scope) { ... } - internal controller [03controller-internal.js]
    NOTICE: $scope which is outside template can be used inside template with {{myVar}}.
  7. controller: function($scope, $element, $attrs, $transclude, otherInjectables) { ... } - usage of $element, $attrs and $transclude [03controller-internal.js]
  8. restrict: 'E | A | C | M' - E-element, A-attribute, C-class, M-comment (EA in most cases)
    replace: true | false - will directive element will be replaced with real HTML tag (default is false)
    [04restrict-e.js, 04restrict-a.js] 
    //template-- <p style="color:green"> </p>
//directive--  <mydir class="pull-right"> </mydir>

//replace: true (will merge directive and template attributes)
//result will be:  <p class="pull-right" style="color:green"> </p>


//replace: false
<mydir class="pull-right"> <p style="color:green"> </p> </mydir>

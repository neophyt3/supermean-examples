Basic Authentication

/lib/factory/basicAuth.js
1. encode 'username:password' string to base64 (john:test => am9objp0ZXN0)
2. send request with header: Authorization: Basic am9objp0ZXN0
3. if auth was successful add 'Basic am9objp0ZXN0' to cookie
